The IUP-Rust Prelude.

This is a prelude module that reexports many of the most common traits, types and functions.

The users can glob import this prelude module to get access to many of the most common traits, types and functions.

use iup::prelude::*;

The prelude is primarily concerned with exporting traits that are so pervasive that it would be obnoxious to import each them every time.
